LIN 177: Computational Linguistics

Lecture video

Lecture capture of in-person class

This lecture continues the topics of context-free grammars, probabilistic context-free grammars, and parsing with CFGs and PCFGs, which were introduced in the previous lecture. We then look at adding features to these grammars (which are called "arguments" in the prolog language), and start the topic of syntactic dependencies.

Homework Assignments

Homework assignments are made avaible to students through canvas. The links below contain the assignments extracted from canvas, but the display differs in minor ways (students would be able to see the normal canvas navigation menus, grading and due date information, etc.)

The meaning of the assignment names eventually become clear to the students (e.g., "wreck a nice beach" -> "recognize speech", etc.)

To complete the homework assignments, students use a custom-built web interface I developed for this course during this review period. The tool allows students to enter a program consisting of a specification for a Finite-State Transducer, issue queries, and see the output of the transducer. Link: http://compling.ucdavis.edu/taufst
